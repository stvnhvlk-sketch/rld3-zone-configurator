# RLD3 Zone Configurator — Home Assistant add-on repository

Home Assistant add-on that runs the visual zone configurator for the
RLD3-Target mmWave presence sensor as an **ingress** sidebar panel.

## Install

1. Home Assistant → **Settings → Add-ons → Add-on Store**.
2. Top-right **⋮ → Repositories** → paste this repo's URL → **Add**.
3. Open **RLD3 Zone Configurator** in the store → **Install** → **Start**.
4. Enable **Show in sidebar** to get the panel.

See [`rld3-zone-configurator/README.md`](rld3-zone-configurator/README.md) for usage.
